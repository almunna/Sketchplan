🧾 Project Setup Instructions
📦 Backend Setup
Open your terminal
1. Navigate to the backend folder:
  cd backend
2. Install Packages:
   npm install
3. Start the server using nodemon:
  nodemon start

💻 Frontend Setup
Open a new terminal window or tab
1. Navigate to the frontend folder:
  cd frontend
2. Install Packages:
   npm install
3. Start the development server:
  npm run dev
